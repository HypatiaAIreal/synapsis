### **🚀 CÓMO USAR EN SYNAPSIS**

```typescript
// 1. Instalar
npm install ./packages/film-methodologies

// 2. Importar
import { FilmMethodologyService } from '@synapsis/film-methodologies';

// 3. Usar
const service = new FilmMethodologyService();
const metodologias = service.selectAdaptiveMethodologies({
  film: { title: 'Parasite', year: 2019, genres: ['thriller'] },
  query: { questionType: 'ideological', depth: 'deep' }
});
```
