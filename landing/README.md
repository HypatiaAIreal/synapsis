# 🎬 SYNAPSIS Landing Page

## 🚀 Landing Page Promocional de SYNAPSIS

Una landing page moderna, elegante y viral para promocionar SYNAPSIS - La revolucionaria plataforma de análisis cinematográfico con IA.

## ✨ Características

### 🎨 Diseño
- **Dark Mode Elegante**: Interfaz oscura con acentos vibrantes
- **Gradientes Dinámicos**: Transiciones de color púrpura a azul
- **Glassmorphism**: Efectos de vidrio esmerilado modernos
- **Animaciones Suaves**: AOS (Animate On Scroll) para elementos
- **Responsive Design**: Perfecta en móvil, tablet y desktop

### 🎯 Secciones
1. **Hero Section**
   - Título impactante con gradientes
   - Estadísticas clave animadas
   - CTAs prominentes
   - Formas flotantes animadas

2. **Cómo Funciona**
   - Proceso de 4 pasos visualizado
   - Cards interactivas con hover effects

3. **Características**
   - Grid de 6 features principales
   - Destacado especial para las 141 metodologías
   - Iconos y badges distintivos

4. **Metodologías**
   - Showcase de las 14 categorías
   - Contadores de metodologías
   - Link para explorar todas

5. **Comunidad**
   - Estadísticas animadas con contadores
   - Features de la comunidad
   - Visualización de alcance global

6. **Testimonios**
   - Cards con ratings de estrellas
   - Testimonial destacado
   - Fotos y credenciales

7. **Planes de Precio**
   - 3 planes: Gratis, Pro, Academia
   - Plan Pro destacado
   - Garantías y badges

8. **FAQ**
   - Preguntas frecuentes organizadas
   - Diseño en grid responsivo

9. **CTA Final**
   - Llamada a la acción potente
   - Cards flotantes animadas
   - Sin fricción (no requiere tarjeta)

### 🔧 Funcionalidades

1. **Navegación Inteligente**
   - Navbar que se oculta al scroll down
   - Smooth scrolling a secciones
   - Mobile menu responsivo

2. **Animaciones**
   - Parallax en shapes del hero
   - Contadores animados
   - Hover effects en cards
   - Partículas flotantes

3. **Interactividad**
   - Modal de video demo
   - Formularios con feedback visual
   - Easter egg (Konami Code)
   - Tracking de eventos

4. **Optimización**
   - Lazy loading de imágenes
   - Debounce en scroll events
   - Service Worker ready (PWA)

## 📁 Estructura de Archivos

```

```

## 🚀 Cómo Usar

1. **Desarrollo Local**
   ```bash
   # Servir con cualquier servidor local
   npx serve landing/
   # O con Python
   python -m http.server 8000
   ```

2. **Personalización**
   - Editar variables CSS en `:root`
   - Modificar textos en `index.html`
   - Ajustar animaciones en `script.js`

3. **Imágenes Necesarias**
   - `app-preview.png` - Screenshot de la app
   - `community-visual.png` - Visual de comunidad
   - `testimonial-[1-3].jpg` - Fotos de testimonios
